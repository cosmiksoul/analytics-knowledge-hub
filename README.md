# Analytics Knowledge Hub

This repository is a curated collection of cheat sheets, manuals, and study materials accumulated over years of study and professional work in data analytics and data science. It is designed to serve as a structured reference for both preparing for technical interviews and refreshing knowledge for daily work tasks.

## Contents

The hub covers a wide range of topics essential for modern analytics:
* **Core Fundamentals:** Mathematics, Probability, Statistics, and Ordinary Differential Equations.
* **Technical Stack:** Deep dives into SQL, Python, R, Excel, and Matlab.
* **Data Engineering & Viz:** Data Warehousing principles and Visualization techniques.
* **Advanced Analytics:** Basic to intermediate materials on Machine Learning and Deep Learning.

## Usage & Disclaimer

These materials are provided "as is." While the fundamental concepts in mathematics and statistics remain timeless, please note that some specific cheat sheets for technical libraries may be outdated as tools evolve. This repository is intended to provide a solid foundation and a starting point for further, more detailed research.
